## Goal

Build a basic classification model using a small dataset — the classic **Iris flower dataset** — to classify flowers into one of three species based on their measurements.

## Key Requirements

- Load and understand a dataset
- Split data into training and testing sets
- Apply a simple classification algorithm
- Evaluate model performance

## Key Skills Practiced

- Data handling with `pandas`
- Supervised learning basics
- Model training and evaluation with `scikit-learn`

---

## Dataset

The **Iris dataset** contains 150 samples of iris flowers, each described by 4 features:

| Feature | Description |
|---|---|
| Sepal Length | in cm |
| Sepal Width | in cm |
| Petal Length | in cm |
| Petal Width | in cm |

Each sample belongs to one of three species: `Setosa`, `Versicolor`, or `Virginica`.

## How It Works

1. Load the Iris dataset directly from `scikit-learn`
2. Explore it with `pandas` (preview, stats, class counts)
3. Split the data into training (80%) and testing (20%) sets
4. Train a **K-Nearest Neighbors (KNN)** classifier
5. Predict on the test set and measure accuracy
6. Run a prediction on a new sample flower
